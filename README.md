# 🐍 Java Snake Game

A classic Snake Game developed in **Java** using **Swing** and **AWT**. The objective is simple: control the snake, eat apples to grow longer, and avoid hitting the walls or your own body.

---

## 📌 Features

- 🎮 Classic Snake gameplay
- 🍎 Randomly generated apples
- 📈 Live score display
- 🚧 Collision detection
- ⌨️ Keyboard controls
- 🖥️ Built with Java Swing GUI
- ⚡ Smooth game loop using Swing Timer

---

## 🛠️ Technologies Used

- Java
- Swing
- AWT
- Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or later
- Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans, VS Code)

### Clone the Repository

```bash
git clone https://github.com/Manoj070037/Snake-Game.git
```

### Navigate to the Project

```bash
cd Java-Snake-Game
```

### Compile

```bash
javac SnakeGame.java
```

### Run

```bash
java SnakeGame
```

## 🎯 Game Rules

- Eat the red apples to grow.
- Each apple increases your score.
- Avoid hitting:
  - The walls
  - Your own body
- The game ends when a collision occurs.

---

## 📷 Gameplay

- Green squares represent the snake.
- Red circle represents the apple.
- Score is displayed at the top-left corner.
- A "Game Over" screen appears when the game ends.
